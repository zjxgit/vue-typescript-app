# vue-typescript-app

> 这个项目起始于在vue加入强类型的想法，基于vue + typescript 使用 .vue 单文件开发脚手架，支持jsx


## 技术栈
- webpack4
- vue
- vue-router
- vuex
- typescript
- jsx
- tslint
- 单vue文件开发
- vue-cli
- ~~eslint~~

## 开始

``` bash

# Install project dependencies
npm install

# serve with hot reload at localhost:3001
npm run dev

# build for production with minification
npm run build

```
